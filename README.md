# Retail Customer Analytics System

## Overview
Analyzed retail customer data of 500 customers across 15 cities in India to uncover customer demographics, revenue trends, and purchasing behavior.

## Tools Used
Python, Pandas, NumPy, Matplotlib, Seaborn, MySQL, Power BI, Excel

## Key Findings
- Total Revenue: Rs 30.77 Lakhs
- Average Purchase Amount: Rs 6,154
- Top City by Revenue: Hyderabad (Rs 2.79L)
- Top Category: Groceries (Rs 5.06L)
- Platinum members spend 10x more than Bronze members
- Male Customers: 51.8% | Female Customers: 48.2%

## Project Structure
- retail_eda.py — Python EDA and 12-chart dashboard
- retail_sql_queries.sql — 12 SQL queries
- retail_dashboard.png — 12-chart visual dashboard
- Retail_Analytics_Dashboard.pdf — Power BI dashboard
